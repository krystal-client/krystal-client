Installation Guide

Important: By downloading and using Krystal Client, you agree to our Terms of Service. Please ensure you comply with server rules when using the client.

1

Install Fabric Loader

Download and install Fabric Loader for your Minecraft version from the official Fabric website. Make sure you also install Fabric API.

2

Download Krystal Client

Click the download button above to get the latest version of Krystal Client. Save the .jar file to a location you can easily find.

3

Locate Your Mods Folder

Open your Minecraft directory and navigate to the "mods" folder. On Windows, press Windows+R, type %appdata%\.minecraft\mods and press Enter.

4

Install the Client

Copy the Krystal Client .jar file into your mods folder. Make sure no other conflicting mods are present.

5

Launch Minecraft

Start Minecraft using the Fabric profile. Krystal Client will load automatically. Press Right Shift to open the client menu in-game.
